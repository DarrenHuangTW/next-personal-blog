2019年4月9號Google的[官方推特](https://twitter.com/searchliaison/status/1115706765088182272)宣布了一組新的Google搜尋指令 – ***before:***和 ***after:***。在這兩個指令後面放上日期，就可以篩選出某個時間之前、某個時間之後、或某兩個時間點之間的搜尋結果。

![google before after search command](https://www.darrenhuang.com/wp-content/uploads/2019/04/google-before-after-search-command.png)
[連結](https://twitter.com/searchliaison/status/1115707059998052354)

## 使用方法\&範例

這個指令的使用方法相當簡單，在搜尋時加上***before:***和***after:***並給一個時間點，時間點必須是年份，或者確切的年月日；***before:***和***after:***可以單獨使用或一起使用。 以下為幾個實測的例子:

```
韓國瑜 before:2017
```
![google before command example 1](https://www.darrenhuang.com/wp-content/uploads/2019/04/google-before-command-example-1-1024x689.png)
搜尋「韓國瑜」加上「before:2017」，搜尋結果為當選市長以前的新聞。


---

```
韓國瑜 after:2018  
```
![google after command example](https://www.darrenhuang.com/wp-content/uploads/2019/04/google-after-command-example-1024x690.png)
   
加上「after:2018」，搜尋結果為韓國瑜市長的近期新聞 


---

```
韓國瑜 after:2018-11-1 before:2018-11-15  
```
![google before and after command example](https://www.darrenhuang.com/wp-content/uploads/2019/04/google-before-and-after-command-example-1024x723.png)
   
加上「after:2018\-11\-1 before:2018\-11\-15 」，搜尋結果為韓國瑜市長在高雄選舉期間的新聞

註: 這組搜尋指令適用於一般搜尋和Google新聞搜尋，上面截圖用Google新聞舉例。

## 日期細節

1. 日期的部分，如果只輸入年份

* before:2018 \=\> 出現2018\-01\-01前的結果
* after:2018 \=\> 出現2018\-12\-31後的結果
* before:2019 after:2017 \=\> 只出現2018年的結果

2\. 用橫槓或斜線都可以

* before:2018\-12\-31 \= before:2018/12/31

3\. 月份或日期可用一個數字或兩個數字表示

* before:2018**\-5**\-17 \= before:2018**\-05**\-17
* before:2018\-10**\-5** \= before:2018\-10**\-05**
* before:2018**\-6\-6** \= before:2018**\-06\-06**

## 其他注意事項

「一般搜尋」和「新聞搜尋」都可以使用這套指令。在「新聞搜尋」中一定會出現該網頁發布的時間，但在「一般搜尋」中，只有Google認為提供網頁發布時間對搜尋者有幫助時才會出現。

![google timestamp](https://www.darrenhuang.com/wp-content/uploads/2019/04/google-timestamp.png)

---

Google會透過各種方式判定一個網頁上的內容被建立的時間，但不能保證一定精準，畢竟網頁可能在發布內容被修改，或者是網站擁有者聲稱的發布日期和實際發布日期不相同。

關於Google如何判定網頁的時間，這裡有篇更詳盡的[官方說明](https://webmasters.googleblog.com/2019/03/help-google-search-know-best-date-for.html)。
