Backlink又稱作Inbound Link (中文好像叫做”反向連結”)，部落格開張到現在已經有兩篇文章都提到了他的重要性: [Search Engines 如何決定網站的排名](http://www.darrenhuang.com/2015/09/how-search-engines-see-your-website.html) 和 [搜尋引擎運算之Google Penguin](http://www.darrenhuang.com/2015/10/google-penguin.html)。了解了解這個概念固然是有益無害，但假若你正為自己的公司做做SEO分析或是想走更深入一點，就需要一個工具來幫你分析哪些網站有放連結到你那裡、他們是誰、他們是好的連結還是壞的連結….等等等。這篇文章要介紹我很喜歡的一個公司, Moz開發的工具: [Open Site Explorer](https://moz.com/researchtools/ose/)!

### 介紹OSE:

OSE可以讓使用者分析: **自己公司(或對手的網站)的backlink，有多少backlinks，這些backlinks來自那些網站，這些網站的authority是高還是低，連結的anchor text為何**…之類的。使用方法也很簡單，把網址放進去給他跑一下就會有一堆數據跳出來了。

Open Site Explorer 有分試用版跟付費版，如果只是要了解這個工具，試用版雖然只能看到前20筆資料(ex. backlinks)，但已足以讓你一覽他的功能。付費版的話可以讓你看到完整的資料，有興趣多加研究的話，可以註冊拿到一個月免費的試用，我就是因為試用期快到才想說趕快寫個文章介紹~

(題外話，換個email註冊就可以再多用30天，有在做這方面實習或是有個小網站的人可以用這奧步，要記得取消就是了。)

[![basic data](http://3.bp.blogspot.com/-GfHMLjgXSYQ/Vj9-6-KnQ0I/AAAAAAAAAKw/4A833YdxwZI/s640/basic%2Bdata.JPG)](http://3.bp.blogspot.com/-GfHMLjgXSYQ/Vj9-6-KnQ0I/AAAAAAAAAKw/4A833YdxwZI/s1600/basic%2Bdata.JPG)

#### Authority

Authority分成page跟domain兩個，domain是整個eztravel.com的分數，page則是根據每個網頁來計算，上圖例子裡的Page Authority計算該”首頁”而非整個網域的分數。

我部落格的domain是blogspot，Google的產品，Authority自然超高，有93/100，但Page 是digitalmktingnote.blogspot.com，只有1/100…

#### Page Link Metrics

* Just Discovered代表最近60天裡，新的連到此網頁的backlinks。
* Established Links指總共有多少連結(6,641)來自幾個不同的網站(237)。

#### Spam Score

這個是Moz認為的17項危險要素，是個很好的指標，如果違反太多可能被Google判定成不自然或是意圖捏造backlink而遭到懲罰，這個需要升級成Pro才看的到，下面列幾個例子:
* **Thin Content**: 網站裡沒有太多的內容，如果都用圖片或影片代替文字告訴讀者這個網站在幹嘛就可能有這項警告，因為搜尋引擎沒法完全解讀文字以外的內容。
* **Anchor Text Heavy Page**: 整個網頁或文章都是超連結連來連去，就跟一堆hashtag的FB貼文一樣討人厭。
* **Site Link Diversity is Low**: 大部分的外部連結都來自少數幾個網站。

### Filter

[![Filter](http://4.bp.blogspot.com/-rM2R46rC5sc/Vj-KmOJHb5I/AAAAAAAAALI/ewOceMRXW-0/s640/Filter.JPG)](http://4.bp.blogspot.com/-rM2R46rC5sc/Vj-KmOJHb5I/AAAAAAAAALI/ewOceMRXW-0/s1600/Filter.JPG)

接下來有幾個分類，Target, Link Source, Link Type:

#### Target:

用我這篇文章做舉例，網址是 http://digitalmktingnote.blogspot.com/2015/11/seo-backlinks-tool-open-site-explorer-ose.html
* this page: 連到這篇文章的外部連結
* this subdomain:  連到digitalmktingnote.blogspot.com/，可能是其他文章的外部連結
* this root domain: 連到blogspot.com，其他部落格都算在裡面

#### Link Source:

* all pages: 包含下面兩種
* only external: 只計算backlinks，從其他domain下的網站連過來的連結
* only internal:  同一個domain下的連結，ex: A文章連到B文章

#### Link Type:

選項包括 link equity, no link equity, only 301, only 302還有可能比較熟悉的 only follow, only nofollow，主要就是要分對方指到你網站的連結是否有把link juice傳遞過來，那些術語以後有機會我研究研究再來介紹。

### Top Pages

這個選項看的是，同一個domain下，哪個網頁最夯。同時，也可以看到該domain哪些網頁正常(HTTP Status: 200)那些有問題 (ex. 404 page not found)。

易遊網第二多inbound link 的網頁是他們的privacy page，而且有好多網頁是有4XX問題的，大概是一些已經過期的活動網頁沒有移除。
[![top page](http://4.bp.blogspot.com/-52Dxh9HMDmE/Vj-cgsa0UPI/AAAAAAAAAMY/O1CNXnYX44Y/s640/top%2Bpage.JPG)](http://4.bp.blogspot.com/-52Dxh9HMDmE/Vj-cgsa0UPI/AAAAAAAAAMY/O1CNXnYX44Y/s1600/top%2Bpage.JPG)

### Anchor Text

Anchor Text可以看到連到易遊網的都是什麼字眼，”**易游网**“, “**易遊網**“, “**[no anchor text]**” (圖)，看有沒有機會聯絡到該網站或文章的作者，請他們把anchor text改成關鍵字或品牌名稱。懷疑遭受Penguin Penalty也可看看是否有[Over Optimized](http://www.darrenhuang.com/2015/10/seo-%e6%90%9c%e5%b0%8b%e5%bc%95%e6%93%8e%e9%81%8b%e7%ae%97%e4%b9%8bgoogle-penguin/)的問題。

### Link Opportunity

這邊有兩個很不錯的功能，第一個是看看誰提到你的網站但是沒有放上連結，一樣看能否聯絡到該作者，下面是易遊網的例子，可以看到chinapost 提了易遊網好幾次卻不放連結過去。

[![link opportunity](http://1.bp.blogspot.com/-A1BZD8wrjhA/Vj-honAW8OI/AAAAAAAAAMw/EHhQ0u6TfbE/s640/link%2Bopportunity%2B.JPG)](http://1.bp.blogspot.com/-A1BZD8wrjhA/Vj-honAW8OI/AAAAAAAAAMw/EHhQ0u6TfbE/s1600/link%2Bopportunity%2B.JPG)
[![link opportunity2](http://3.bp.blogspot.com/-ThaERy959oE/Vj-homNGpCI/AAAAAAAAAMs/Ed6meBnUj34/s640/link%2Bopportunity2.JPG)](http://3.bp.blogspot.com/-ThaERy959oE/Vj-homNGpCI/AAAAAAAAAMs/Ed6meBnUj34/s1600/link%2Bopportunity2.JPG)

第二個是看看那些文章有連結連到對手的網站們卻沒有連到你的。我用燦星旅遊(startravel.com.tw)和雄獅旅遊(liontravel.com.tw)做比較，發現有222個網頁有同時提到他們倆卻沒有提到易遊網(或者是有提到但是連結掛掉)。

### 結論:

Open Site Explorer真的是個很不錯的工具，也算不上複雜。有一點必須強調的是，它畢竟是個第三方工具，Moz看得跟Google或Bing看到的不一定一樣! 上面的資料與不敢保證100%正確，像我在這篇文章前面提到易遊網的時候超連結過去，但是好幾個小時過後Just Discovered卻沒有偵測到我這裡有一個易遊網的外部連結。

不過，還是希望大家有在這篇學到東西，下次被老闆要求做backlinks analysis, off-site SEO audit，或是想看看自己網站表現如何的時後，才不會不知道從何下手!
