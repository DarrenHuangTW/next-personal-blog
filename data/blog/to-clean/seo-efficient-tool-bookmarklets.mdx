成為一個SEO從業人員後，三不五時就需要去看看Source Code、使用”site:”, “cache:”之類的指令為網站做健診，這些步驟也許不是特別花時間，但是如果有個工具可以讓你一點就可以檢查某個網頁每張圖的Alt Tag、一點就可以看到該網頁cache的樣子、甚至一點就可以讓你看到該頁面的Title, Description, 還有其他Meta Tags，絕對是省下很多時間! 這篇文章要分享幾個我頗常用也滿喜歡的Bookmarklets，希望能對SEOs們有所幫助!

## 什麼是Bookmarklet?

Bookmarklet的中文叫做小書籤，和一般的書籤不一樣的地方是，**它是一串JaveScript而不是一個URL**。舉例來說，工程師們可以寫一個指令: 告訴我當前網頁的rel canonical tag是什麼，把這個指令放進瀏覽器的書籤列後，以後要檢查網站時就只要點一下該小書籤，就會直接告訴你答案了!

## 使用方法

小書籤的好處就是使用起來超級簡單，就算自身不會coding，只要找到有人分享(像這篇文章)，把連結拖到書籤列就可以方便以後使用了。

## 10個SEOs常用小書籤

下面列出幾個我個人最常用的功能，包括檢查當前頁面的meta robots tag, canonical tag，Wayback Machine，Google Page Speed Insights，開啟Open Site Explorer…等，屁話不多說，一起動手來試試看吧!

### 1\. 檢查當前網站有多少頁被index

> Bookmarklet: [Site Search(site:\+domain)](javascript:location.href='http://www.google.com/search?q=site:'+location.host)

在Google輸入site:www.example.com來找出某網站有多少網頁被Google索引，這個指令對多數SEOs絕對非常熟悉，而這個小書籤讓你不管在該網站的哪一個網頁，都可以一點就搜尋Site\+domain而不用複製貼上來來去去。

![site search operator](http://www.darrenhuang.com/wp-content/uploads/2016/06/site-operator.jpg)

### 2\.  檢查當前網頁是否被index

> Bookmarklet: [Info Search(site:\+current page)](javascript:location.href='http://www.google.com/search?q=info:'+location.href)

這個是我本篇要介紹的小書籤裡面唯一一個自己寫的XD 主要用來檢查當前的頁面是否已經被Google Index。假設你現在在<https://moz.com/blog>，用這個書籤可以讓你看到該頁是否已被索引，而不會出現/blog下的其他頁面。

![info operator](http://www.darrenhuang.com/wp-content/uploads/2016/06/info-operator.jpg)

 

### 3\. 檢查當前網頁的Cache版本

> Bookmarklet: [Google Cache](javascript:location.href='http://www.google.com/search?q=cache:'+location.href)

Crawl / Index / Cache之間的概念就不再多敘述咧，簡單來說這個小書籤可以讓你看到當前網頁在Google的眼裡是長甚麼樣子，在網頁內容有更動後，你可能會常常使用它來看看Google是否已察覺到這個改變。

![google cache](http://www.darrenhuang.com/wp-content/uploads/2016/06/google-cache-version.jpg)

 

### 4\. 開啟Open Site Explorer檢查當前網頁

> Bookmarklet: [用OSE查看當前網頁資料](javascript:location.href='http://www.opensiteexplorer.org/links?site='+location.href)

之前寫過一篇關於[Moz的這個工具Open Site Explorer](http://www.darrenhuang.com/2015/11/seo-backlinks-tool-open-site-explorer-ose.html)，利用此小書籤可以直接開啟OSE然後看看現在所在頁面的一些基本資料(DA/PA, inbound links…等等)，對有Moz Pro的人來說非常好用!

![moz open site explorer](http://www.darrenhuang.com/wp-content/uploads/2016/06/open-site-explorer-1024x507.jpg)

 

### 5\. 檢查當前頁面的Canonical Tag

> Bookmarklet: [Check the canonical tag](javascript: (function () { var url = document.location; var links = document.getElementsByTagName('link'); var found = 'none'; for (var i = 0, l; l = links[i]; i++) { if (l.getAttribute('rel') == 'canonical') { found = l.getAttribute('href'); break; } } alert(found); })();)

一點就會跳出一個小框框告訴你目前所在頁面有沒有rel canonical tag，有的話是canonicalize到哪裡去， \< link href\=”http://www.darrenhuang.com/” rel\=”canonical” /\>。

![check canonical tag](http://www.darrenhuang.com/wp-content/uploads/2016/06/check-canonical-tag.jpg)

 

### 6\. 檢查當前網頁的meta robots tag

> Bookmarklet: [Check meta robots tag](javascript: (function () { var url = document.location; var links = document.getElementsByTagName('meta'); var found = 'none'; for (var i = 0, l; l = links[i]; i++) { if (l.getAttribute('name') == 'robots') { found = l.getAttribute('content'); break; } } alert(found); })();)

這個小書籤會檢查當前網站的meta robots tag，會跳出一個框框和你說robots meta tag裡面是否有index/noindex, follow/nofollow, noodp, noydir, nocache, noarchive….等等指令。

![check meta robots tag](http://www.darrenhuang.com/wp-content/uploads/2016/06/check-meta-robots-tag.jpg)

 

### 7\. 用Wayback Machine看看當前網頁以前長得怎樣

> Bookmarklet: [Wayback Machine](javascript:(function(){%20window.open('http://web.archive.org/web/*/'+location.host)})();)

Wayback Machine是可以看到一個網頁以前的模樣\*，這個小書籤可以讓你一鍵進到Wayback Machine分析完目前網頁的結果，然後選日期看看該網頁在不同時期的樣子。  
(\*註: 不是每個網頁都會被存取的!)

![wayback machine](http://www.darrenhuang.com/wp-content/uploads/2016/06/wayback-machine-1024x502.jpg)

 

### 8\. 檢查當前網頁的Page Speed Insights

> Bookmarklet: [Google Page Speed Insights](javascript:(function(){%20window.open('http://developers.google.com/speed/pagespeed/insights/?url='+encodeURIComponent(location.href))})();)

有點經驗的人對這個工具應該不陌生，可以分析網頁的讀取速度，而此小書籤讓你不用又開新分頁又搜尋又複製貼上，一鍵搞定!

![Google Page Speed Insights](http://www.darrenhuang.com/wp-content/uploads/2016/06/google-page-speed-insights-1024x503.jpg)
